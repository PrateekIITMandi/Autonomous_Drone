DP Project : Autonomous Drone for Disaster Management and Surveillance
This autonomous drone is designed specifically to find people trapped in disaster zones and geotag them ; 
can send information to the control room, so that rescue teams can act accordingly. 
It operates within a predefined area, flying autonomously without manual control.

Contents of the folders :
cad : Contains CAD design of the Drone (to be added) ;
codes : relevant codes for geotagging, and autonomous navigation (to be added) ;
training dataset generation : codes for training dataset generation, yolov8l-pose.onnx used for human detection
fine tuning : code for fine tuning ;
docs : Midsem Report, Poster, and Journal Paper (to be added) ;
materials : Materials used for the Drone with their Cad models (to be added)
